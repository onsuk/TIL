# VS Code Useful Shortcut

**Mac** 개발환경 기준으로 개인적으로 유용했던 VS Code 단축키를 정리하는 글이다.

- **들여쓰기 자동 정렬** : `option + command + f`
- **전체 검색** : 원하는 단어 선택(드래그), `shift + command + f, return`
- **File Explorer** : `shift + command + e`
- **한줄 복사 & 붙이기** : `shift + option + 방향키`
- **터미널 창** : ``control + ` ``
