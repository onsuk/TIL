# 데이터 처리(Data Processing)

## 종류
- 일괄처리 시스템 - 데이터를 모아서 한꺼번에 처리하며 효율이 좋다.
- 다중 프로그래밍 시스템 - 하나의 CPU를 이용, 여러 개의 프로그램을 동시에 실행한다.
- 시분할 시스템 - 여러 프로그램을 약간의 시간차를 두고 번갈아 실행하여, 동시에 실행되는 느낌을 준다.
- 다중 처리 시스템 - 여러 개의 CPU를 이용, 여러 개의 프로그램을 동시에 실행한다.
- 실시간 처리 시스템 - 데이터 처리의 요구가 발생한 즉시 처리한다
- 다중 모드 처리 시스템 - 일괄, 시분할 처리 시스템 등의 여러 시스템들을 중복 사용하는 시스템
- 분산 처리 시스템 - 여러 개의 컴퓨터를 연결하여 하나의 작업을 처리.
